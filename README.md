## 🎥 Video Setup

### Required Video
Place your hero background video at: `videos/perfume-hero.mp4`

### Video Recommendations
- **Format**: MP4 (H.264 codec)
- **Resolution**: 1920x1080 (Full HD) or higher
- **Duration**: 10-30 seconds (looping)
- **Content Suggestions**:
  - Perfume bottle rotating elegantly
  - Fragrance mist in slow motion
  - Luxury perfume bottles on display
  - Abstract fluid/smoke movements
  - Close-up of perfume being sprayed

## 🚀 How to Use

1. **Install XAMPP** and start Apache server
2. **Place files** in `C:\xampp\htdocs\scentscape\`
3. **Add video** to `videos/perfume-hero.mp4`
4. **Open browser** and navigate to `http://localhost/scentscape/`
5. **Toggle theme** using the moon/sun icon in navigation
